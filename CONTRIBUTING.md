# Руководство по внесению вклада

## Переменные окружения

Часть настроек проекта берётся из переменных окружения. Чтобы их определить, создайте файл .env рядом с main.py и запишите туда данные в таком формате: ПЕРЕМЕННАЯ=значение.

Доступны следущие переменные:

Google Sheets

GOOGLE_SHEETS_ID=ID_вашей_таблицы_из_адресной_строки

Telegram

TG_BOT_TOKEN=токен_от_BotFather TG_CHAT_ID=ID_канала_или_чата (например, -100...)

ВКонтакте

VK_GROUP_ID=ID_группы_без_минуса VK_USER_TOKEN=ваш_личный_токен_с_правами_wall VK_GROUP_KEY=токен_сообщества (если используется)

Одноклассники

OK_PUBLIC_KEY=публичный_ключ_приложения OK_SECRET_KEY=секретный_ключ_приложения OK_ACCESS_TOKEN=токен_доступа_аккаунта OK_SESSION_SECRET=секретный_ключ_сессии OK_GROUP_ID=ID_группы_в_ОК
